Diggaren

Hem Radio Spotify API Dokumentation

Grupp 12 har tagit fram tjänsten Diggaren. Diggaren vänder sig till alla de personer som regelbundet använder radio men även Spotify för att lyssna på musik. Tjänsten låter användaren att på ett visuellt tilltalande sätt bläddra bland ett urval av radiokanaler, se vad som spelas nu samt vad som nyss spelades. Tjänsten låter även användaren hitta låtarna på Spotify, så att man kan lyssna på favoritlåtarna man hör på radio - om och om igen.
